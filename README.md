# Analyse exploratoire et prédictive des données Wine – Python & scikit-learn

##  Objectif du projet
Réaliser en **Python** une **analyse exploratoire et prédictive** du jeu de données **Wine**, afin d’évaluer l’impact de différentes techniques de traitement et de modélisation sur la **performance de prédiction**.

Le projet couvre :
- l’analyse statistique et la corrélation des variables,
- la gestion des données manquantes,
- des méthodes de classification,
- la réduction de dimension,
- l’interprétation des résultats dans un **rapport** et une **présentation**.

---

##  Jeu de données
- **Wine Dataset** (UCI Machine Learning Repository)
- Données chimiques et physiques de vins classés par type

---

##   Technologies et bibliothèques
- Python
- pandas
- numpy
- matplotlib / seaborn
- scikit-learn
- Jupyter Notebook

---

##   Méthodologie
1. Exploration des données (EDA)
2. Analyse de corrélation entre variables
3. Gestion des données manquantes :
   - suppression
   - imputation
4. Mise à l’échelle des données
5. Classification :
   - k-NN
   - Régression logistique
   - Arbres de décision
6. Réduction de dimension :
   - PCA
7. Évaluation des performances

---

##   Fonctionnalités
- Visualisation des distributions et corrélations
- Comparaison de différentes stratégies de traitement des données
- Entraînement et évaluation de modèles de classification
- Analyse de l’impact de la réduction de dimension
- Comparaison des performances (accuracy, confusion matrix)

---

##   Résultats
- Amélioration ou dégradation des performances selon :
  - la méthode d’imputation
  - le modèle de classification
  - l’utilisation ou non de PCA
- Analyse critique des résultats obtenus
